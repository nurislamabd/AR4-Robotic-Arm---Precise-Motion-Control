# AR4-Robotic-Arm---Precise-Motion-Control

- Developed and debugged MATLAB-based control algorithms for AR4 robotic arm joint positioning and speed estimation, achieving precise position tracking with negligible error and real-time RPM computation
- Implemented motor initialization and multi-joint sequencing enabling reliable startup calibration across all robot axes
- Designed and validated position/speed control for all motors by deriving empirical frequency-to-RPM relationships and enabling programmable motion control
- Currently working on kinematics and buidling gripper for the robot.
